      Alpine Linuxとは

      • muslとBusyBoxをベースとしたLinuxディストリビューションの1つ.
      • セキュリティ・シンプル・リソース効率を重視するパワーユーザー向けに設計されている.
      • とても軽量でセキュアなのでDockerのイメージ作りに使われる.
      • 公式サイト: https://alpinelinux.org

      簡単な使い方

      # gitをインストール
apk add git
