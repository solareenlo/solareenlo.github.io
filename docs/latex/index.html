    LaTeX

    LaTeXとは

    • テキストベースの組版処理システムのこと.
    • 単一のアプリケーションではなく, 膨大な数の実行形式ファイル, マクロ, フォントファイル, 周辺ユーティリティの集合体.
    • 学術機関における論文執筆ツールとしてよく使われている.
    • GitHubリポジトリ: https://github.com/latex3/latex3
